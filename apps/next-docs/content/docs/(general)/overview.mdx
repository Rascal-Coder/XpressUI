## 什么是 Xpress UI？

Xpress UI 组件使用 [shadcn/ui](https://ui.shadcn.com/) 的 CSS 变量，并设计为与现有基础组件完美配合。

与核心 shadcn/ui 基础组件类似，Xpress UI 组件本质上是一组构建块，可用于创建更复杂的组件。它们涵盖了广泛的使用场景，旨在减少样板代码，同时不牺牲灵活性和响应式设计。

虽然 shadcn/ui 主要专注于封装 Radix UI 的基础组件，但 Xpress UI 旨在成为一个更全面的组件库，可用于构建更复杂的应用程序。这些组件封装了来自各种无头库的更复杂逻辑，以提供更无缝的体验。
